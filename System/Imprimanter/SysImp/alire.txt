J’ai réussi de réaliser la fonction de Client, Serveur et Imprimante. Quand client tape la commande ¨./mpr -P imp1 référence¨ qui envoie le message d’imprimer le fichier de la référence relatif ou absolue que le serveur peut recevoir par le tube nommé. Tout d’abord, le serveur crée un tube nommé pour communiquer avec les clients, trois tubes nommés pour communiquer avec les imprimantes. Quand le serveur reçoit le message du client, il vérifie si le client a droit de lire ce fichier et opérer ce fichier. S’il a le droit, le serveur envoie le message à l’imprimante, l’imprimante imprime le fichier. Sinon, le serveur affiche l’erreur d’imprimer.
